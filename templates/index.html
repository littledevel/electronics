Inventory

This website is a simple inventory for materials like electronic items or screws.

The usage is simple, it creates a simple text based json "db" like file and stores all items inside it.

You can simply store it in a git repo and keep it updated with simple commits.

It does not have to connect to any other database, or any kind of other connection.

How it works

Simply populate the "categories.json" file with the kinds of items you want to store, and you are ready to go.

Hit the "Items" menu to add,remove update items in your "db".

A bit of warning!

Since the purpose of this simple app is just to facilitate a small inventory, it is not designed to create backups, store items safely or anything like that. If you delete something from the file and you do not commit your changes, your item is lost forever.

Enjoy !
